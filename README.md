# AI Study Planner

AI-powered study planning platform built with TanStack Start, Supabase, and Lovable.

## Features
- Authentication
- Google Login
- Subject Management
- Exam Management
- AI Study Plan Generation
- Progress Tracking
- Dark Mode

## Live Demo
https://ai-learn-plan.lovable.app

## Tech Stack
- TanStack Start
- React
- TypeScript
- Supabase
- Tailwind CSS
